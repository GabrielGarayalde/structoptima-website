Column Design

The following program allows yields the optimal member size for a simply supported beam column under combined axial compression and major axis bending. The program takes as input the values of the slider and calculates the most efficient member size for in-plane Major axis bending. It assumes that the member is fully laterally restrained.

Effective Length in x (Le_x): 7.5 [m]
Axial Compression (N*): 7.5 [kN]
Design bending moment (Mxi*): 30 [kNm]
